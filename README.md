Vue-Cli-Init-Revised
================================

1. A little bit change in vue cli to fit for any template address


使用
================================

1. 首先全局安装
```
git clone

cd vue-init-cli-revised

npm install -g

```

2. 输入命令创建新项目
    - templateAddress是template的地址，如果有branch在最后添加#branch名字
    - projectName是项目名字
```
vt init templateAddress projectName
```
